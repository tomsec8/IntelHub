Privacy Policy – IntelHub

Effective Date: June 25, 2026

IntelHub is an Open Source Intelligence (OSINT) workstation extension designed to help researchers aggregate public tools. Because we serve the investigative community, transparency regarding data transmission is our highest priority.

1. Data Collection & Processing

IntelHub does not actively collect, record, or track your personal data, browsing history, or analytics. The extension operates locally on your machine. However, to provide OSINT functionalities, specific features intentionally transmit user-provided data (e.g., URLs, usernames, or images) to specific third-party services as detailed below.

Local Processing: Features like Metadata Extraction (EXIF, PDF, Office) and File Hashing are performed strictly locally on your device without communicating with any external servers.

2. Third-Party Data Transmission

If you choose to use the following specific features, IntelHub will transmit your input to the corresponding third-party services:

  • Reverse Image Search (File Upload): When you choose to upload an image from your computer to run a multi-engine reverse search, the image is temporarily uploaded via POST request to a public anonymous file host (catbox.moe) to generate a public URL. This URL is then passed to search engines (Google, Yandex, Bing, TinEye). Users are presented with an explicit consent warning before any image leaves the device.
  • Site Analyzer (URL Unshortening): When you choose to unshorten a link, the extension securely resolves the final destination locally on your device by following HTTP headers. No third-party proxy or external URL unshortener is used.
  • Quad-Engine AI Assistants: When you activate and type into the AI tab, your queries are transmitted directly to the respective AI providers (Google AI, Brave Search, Phind, AndiSearch) via embedded iframes.
  • DNS & OSINT Lookups: Resolving IP addresses in the Site Analyzer relies on the Google DoH API (dns.google). Other investigation buttons open new tabs directly to third-party services (e.g., VirusTotal, W3Techs, WHOIS) passing your target domain as a URL parameter.
  • Official Tools List: The extension occasionally fetches its internal tool directory directly from our official GitHub repository (raw.githubusercontent.com).

3. Permissions Explanation

  • activeTab: Used to detect the currently active tab when the user explicitly clicks features like "Save Page Offline" or "Site Analyzer."
  • storage & unlimitedStorage: Used to save user preferences, custom graph nodes, and UI configurations locally in your browser.
  • scripting: Required to dynamically inject CSS/JS into the AI iframes to provide a clean UI interface.
  • declarativeNetRequest: Used strictly to allow the embedding of the aforementioned AI search engines within the extension interface.
  • downloads: Used exclusively to save generated reports, graphs, or offline evidence HTML files directly to your local machine.
