IntelHub is a Chrome extension that helps users access and organize Open Source Intelligence (OSINT) tools directly in their browser. Your privacy is important to us. This page outlines what data we collect and how we use it.

1. Data Collection

IntelHub does not collect, store, or transmit any personal data, browsing history, or user content.

All metadata analysis (images, PDF, Office files) is processed locally on your device and not uploaded to any server.

2. Permissions Explanation

  • activeTab: Used to detect the currently active tab when the user interacts with it (e.g., for reverse image search).
  • storage: Used to save user preferences such as favorites and selected engines locally.
  • scripting: Required for injecting scripts when the user chooses to extract image URLs.
  • host_permissions: Needed for fetching the tool list from GitHub or analyzing site content.

None of these permissions are used to track or profile users.

3. No Third-Party Sharing

IntelHub does not share any user data with third parties. It does not use cookies or analytics services.

4. Security

All processing is done locally on your device. The only external request is to fetch the tool list from the official GitHub repository:

https://github.com/tomsec8/IntelHub

5. Changes to This Policy

Any changes to this privacy policy will be published here and reflected in the GitHub repository and the Chrome Web Store listing.

6. Contact

If you have any questions or concerns about this policy, feel free to open an issue on GitHub:
