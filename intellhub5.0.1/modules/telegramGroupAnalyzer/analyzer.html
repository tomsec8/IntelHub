📋 How to Use This Tool

  1. Export from Telegram Desktop: Open the desired group/chat → Click the three dots (⋮) → 'Export chat history'.
  2. Select Settings: Make sure to check 'Photos', 'Videos', 'Files', etc. The format must be 'HTML'.
  3. Compress the Folder: Once the export is complete, find the folder on your computer. Right-click it and compress it into a single .zip file.
  4. Upload the .zip File: Click the button below or drag & drop the .zip file you just created.
